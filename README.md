androidstudio4maccheatsheet
===========================

Android Studio for Mac Cheat sheet lists all most useful keyboard shortcut by default in Android Studio.

### Quick fixes / Must knows

| Action           | Keys          | Context       | 
| ---------------- | ------------- |:-------------:|
| Quick fix        | ⌥ ENTER       |               | 
| Organize import  | ⌃ ⌥ O         |               |  
| Indent lines     | ⌃ ⌥ I         |               |
| Reformat code    | ⌥ ⌘ L         |               | 
| New Class        | ⌘ N           | Project  tab  | 
| New Class        | ⌘ N           | Java File     | 
| Show docs        | ⌃ Q           |               | 
| Command look-up  | ⇧ ⌘ A         |               | 

### Navigation
| Action           | Keys          | Context       | 
| ---------------- | ------------- |:-------------:|
| Open class       | ⌘ O           |               | 
| Open file        | ⇧ ⌘ O         |               |  
| Go to line       | ⌘ L           |               |

### Refactoring

| Action           | Keys          | Context       | 
| ---------------- | ------------- |:-------------:|
| General refactor | ⌃ T           |               | 
| Rename           | ⇧ F6          |               |  
| Extract Variable | ⌥ ⌘ V         |               |
| Extract Field    | ⌥ ⌘ F         |               |
| Extract Constant | ⌥ ⌘ C         |               |
| Extract Parameter| ⌥ ⌘ P         |               |
| Extract Method   | ⌥ ⌘ M         |               |
| Surround with    | ⌥ ⌘ T         |               |
